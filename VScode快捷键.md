# VScode快捷键

|       按键        |          作用           |
| :---------------: | :---------------------: |
|   Alt+ Up/Down    |     当前行上下移动      |
|     alt+光标      |    点的位置出现光标     |
|      Ctrl+.       |        快速修复         |
|     Ctrl+[ /]     |          缩进           |
|      Ctrl+D       |  匹配当前选中的词汇/行  |
|    Ctrl+G 行号    |        快速跳转         |
|      Ctrl+L       |       选中当前行        |
|      Ctrl+P       |     打开最近的文件      |
| Ctrl+Shift Enter  | 在当前行上插入新的一行  |
|   Ctrl+Shift K    |       删除当前行        |
|      Ctrl+W       |        关闭窗口         |
|    Shift+光标     |        选中多行         |
|    Shift+Alt A    |       块区域注释        |
|    Shift+Alt F    |       格式化代码        |
| Shift+Alt Up/Down | 在当前行下/上复制当前行 |
